## Brand API endpoint

```bash Request
curl "https://api.logokit.com/brands/<identifier>" \
     --header "Authorization: Bearer: <secret-api-token>"
```

### Authentication

Brand API endpoints are authenticated using Bearer token authentication. You must
provide a valid API token in the `Authorization` header of your request. This token
is used to verify your identity and grant access to the API. Make sure to keep
this secret API token secure and do not share it publicly. You can find yours in
[your account](https://logokit.com/account/api-tokens).

```http
Authorization: Bearer: <secret-api-token>
```

### Parameters

<ResponseField name="identifier" type="string" required>
  The domain or stock ticker for which you want to retrieve the logo. For
  example, `apple.com` or `AAPL`. We support all major stock exchanges
  worldwide, including NYSE, NASDAQ, TYO, LSE, and more.
</ResponseField>

## Rate Limiting

Similar to the Logo API, the Brand API implements rate limiting based on user subscription plans, with access only for paid users.
To get more information about the rate limits, please refer to the [pricing page](https://logokit.com/pricing).
