## Logo API

The Logo API is available for both free and paid plans. However, the logos served with a free API token are limited to 64x64 pixels. If you need higher resolution logos, you can upgrade to a paid plan.

<Info>
  Free plan users are limited to 64x64px logos. If you need higher resolution
  logos, please [upgrade here](https://logokit.com/pricing).
</Info>

## Brand API

<Info>
  Free plan users do not have access to the Brand API. If you need access to the
  Brand API, please [upgrade here](https://logokit.com/pricing).
</Info>
